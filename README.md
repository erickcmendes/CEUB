# 📊 Repositório de Projetos Acadêmicos - CEUB (Ciência de Dados e Machine Learning)
Este repositório reúne os projetos desenvolvidos ao longo da trajetória acadêmica, com foco em **qualidade técnica, rigor metodológico e aplicação prática em nível profissional**.


---

## 🎯 Objetivo

Consolidar, versionar e documentar projetos acadêmicos com padrão profissional, demonstrando:

- Capacidade analítica e pensamento crítico  
- Aplicação de técnicas modernas de **Ciência de Dados**  
- Organização e boas práticas de desenvolvimento  
- Evolução contínua ao longo dos semestres  

---

## 🗂️ Estrutura do Repositório

O repositório está organizado por disciplina. Cada pasta representa uma matéria cursada:
```
/
├── disciplina_1/
│   ├── projeto/
│   ├── dados/
│   ├── notebooks/
│   └── README.md
│
├── disciplina_2/
│   └── ...
```

Dentro de cada disciplina, serão incluídos:

- 📁 **Projeto principal** da matéria  
- 📊 **Bases de dados** (quando aplicável)  
- 📓 **Notebooks exploratórios**  
- 📄 **Documentação detalhada**  

---

## 🚀 Escopo dos Projetos

Os projetos aqui presentes abrangem diversas áreas, incluindo:

- Análise Exploratória de Dados (EDA)  
- Modelagem Estatística  
- Machine Learning  
- Engenharia de Dados  
- Visualização de Dados  
- Automação e pipelines de dados  

Todos os projetos são desenvolvidos com foco em **aplicabilidade real**, simulando cenários e desafios do mercado.

---

## ⚙️ Boas Práticas

Este repositório segue princípios de desenvolvimento profissional:

- Versionamento com Git  
- Organização modular de código  
- Documentação clara e objetiva  
- Reprodutibilidade dos experimentos  
- Uso de ambientes virtuais (`venv`)  
- Padronização de dependências (`requirements.txt`)  

---

## 📈 Evolução Contínua

Este repositório é vivo e será atualizado constantemente ao longo dos semestres, refletindo:

- O aprofundamento técnico nas disciplinas  
- A evolução na complexidade dos projetos  
- A incorporação de novas ferramentas e tecnologias
